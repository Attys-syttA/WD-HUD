# Acceptance Checklist

Last updated: 2026-07-04

- [ ] Build succeeds.
- [ ] Tests succeed.
- [ ] Security scan succeeds.
- [ ] App starts.
- [ ] HUD is visible.
- [ ] HUD is glass-like and translucent.
- [ ] Time is visible.
- [ ] CPU usage is visible.
- [ ] RAM usage is visible.
- [ ] GPU value is a number or `N/A`.
- [ ] CPU temperature is a number or `N/A`.
- [ ] GPU temperature is a number or `N/A`.
- [ ] Multiple-GPU selection is handled.
- [ ] Missing sensor values do not crash the app.
- [ ] Settings are saved and restored.
- [ ] Always-on-top works.
- [ ] Click-through can be enabled.
- [ ] Outbound network block script can create a rule.
- [ ] Verify script can detect the rule.
- [ ] No updater exists.
- [ ] No telemetry exists.
- [ ] No analytics exists.
- [ ] No unnecessary dependency is present.
